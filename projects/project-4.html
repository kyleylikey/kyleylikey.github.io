Project Kolehiyo

Developed with: 

AI Chatbot for Incoming College Students

Project Kolehiyo is an AI chatbot designed to assist incoming college students in the Philippines. The chatbot asks a series of pre-made background questions and then provides information about various courses, helping students make informed decisions about their education.

Key Features

  • OpenAI API: The chatbot is powered by OpenAI's gpt4.1 nano model
  • Personalized Suggestions: Users receive tailored college program suggestions
  • Responsive Design: The website works on both desktop and mobile devices

For more details about Project Kolehiyo, check out the public repository I made or visit the deployed website.

Take note that the website is deployed on a free-tier server, so it may take a few seconds to load.
